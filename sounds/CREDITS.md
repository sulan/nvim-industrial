## Workshop pack sounds

The following sounds are derived from the **"68 Workshop Sounds"** pack:

| File | Source sound | Event |
|------|--------------|-------|
| `hammer.mp3`       | workshop - clink1.wav       | Typing in insert mode |
| `drill_start.mp3`  | workshop - drill short 1.wav | Enter insert mode |
| `rivet.mp3`        | workshop - clink thud.wav   | Open file |
| `factory_bell.mp3` | workshop - jingle.wav       | Neovim startup |
| `chainsaw.mp3`     | workshop - loud clatter.wav | Close buffer |
| `grinder.mp3`      | workshop - scrape3.wav      | Text change (normal mode) |
| `gear_crank.mp3`   | workshop - ratchet1.wav     | Enter command mode |

**Pack:** 68 Workshop Sounds
**Author:** Celianfrog
**Source:** https://opengameart.org/content/68-workshop-sounds
**License:** CC0 1.0 Universal — https://creativecommons.org/publicdomain/zero/1.0/

## Synthesized sounds

The following sounds were generated programmatically using
[ffmpeg](https://ffmpeg.org/) audio synthesis filters and are dedicated to
the public domain by the nvim-industrial contributors:

| File | Description | Method |
|------|-------------|--------|
| `steam_hiss.mp3`        | Pneumatic steam release (0.5s) | White noise, bandpass filter, short fade |
| `explosion.mp3`         | Industrial blast (0.8s)        | Brown noise, bass boost, limiter |
| `factory_shutdown.mp3`  | Turbine spin-down (1.5s)       | Exponentially decaying frequency chirp, sub-harmonic, amplitude wobble |
